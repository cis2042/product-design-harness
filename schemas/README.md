# Schemas

These schemas define machine-checkable Product Design Harness outputs.

| File | Use |
|---|---|
| `actor-boundary.schema.json` | Shared actor boundary reused by product brief, review result, and context pack. |
| `claim-analysis.schema.json` | Claim graph and critical path. |
| `claim.schema.json` | Claim analysis item. |
| `context-pack.schema.json` | Handoff after Continue. |
| `council-input.schema.json` | Join-barrier input for council synthesis. |
| `decision-rule.schema.json` | Machine-readable UX3 decision rule. |
| `evidence-request.schema.json` | Request contract for external evidence retrieval. |
| `evidence-response.schema.json` | Response contract for external evidence retrieval. |
| `evidence.schema.json` | Evidence receipt with provenance and freshness. |
| `human-decision.schema.json` | Human-owned decision record. |
| `knowledge-record.schema.json` | Durable knowledge classified by time, epistemic state, space, and authority. |
| `ontology.schema.json` | Machine-readable UX3 ontology. |
| `product-brief.schema.json` | Machine-readable intake. |
| `review-result.schema.json` | Canonical final result for every review mode. |
| `reviewer-verdict.schema.json` | Individual reviewer output per round; summarized into the canonical result's lane reviews. |
| `session-config.schema.json` | Working-language session configuration. |
| `uncertainty-review.schema.json` | Evidence tier, risk, reversibility, and proof bar. |
