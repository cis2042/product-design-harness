# Examples

| File | Purpose |
|---|---|
| `quick-gate-review.md` | Small reversible change. |
| `quick-gate-review.json` | Machine-checkable Quick Gate result. |
| `quick-context-pack.json` | Machine-checkable handoff for the Quick Gate continue result. |
| `live-coding-product-brief.json` | Product brief for a complete live-coding brief -> review -> context-pack chain. |
| `live-coding-review.json` | Machine-checkable live-coding review with a legitimate `continue` verdict. |
| `live-coding-context-pack.json` | Machine-checkable live-coding implementation handoff for the approved bounded change. |
| `stop-reframe-review.md` | Human-readable stop_reframe walkthrough. |
| `stop-reframe-review.json` | Machine-checkable stop_reframe result: t0 evidence forces the stop. |
| `ux3-council-input.json` | Machine-checkable council input: claim graph, blind first-round verdicts, challenge round, uncertainty review. |
| `feature-gate-review.md` | End-to-end example of a vague feature request becoming a gate review and handoff. |
| `standard-gate-review.json` | Machine-checkable Standard Gate result. |
| `ux3-council-review.md` | Extended UX3 council review. |
| `ux3-council-review.json` | Machine-checkable Council result. |
| `human-decision.json` | Accountable decision record used by the Council example. |
| `evidence-response.json` | Evidence receipts with provenance and freshness. |
