# Prompts

| File | Use |
|---|---|
| `start-review.md` | Choose mode and start the harness. |
| `mode-selector.md` | Select Quick Gate, Standard Gate, or UX3 Council, or route to context-pack handoff. |
| `intake.md` | Turn a vague idea into a reviewable hypothesis. |
| `gate-review.md` | Run the full UX3 review. |
| `claim-analysis.md` | Break broad product ideas into testable claims. |
| `ux3-council.md` | Run stage, claim, flow, uncertainty, and council review. |
| `red-team.md` | Challenge an overly convincing product plan. |
