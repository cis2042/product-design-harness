# Agents

These files define reviewer roles.

`registry.json` is the machine-readable role registry. The Markdown files are
the human-readable instructions.

| File | Role |
|---|---|
| `user-flow-reviewer.md` | Checks actor, problem, affected people, and user reality. |
| `evidence-flow-reviewer.md` | Checks proof, measurement, failure criteria, and traceability. |
| `business-flow-reviewer.md` | Checks value, cost, distribution, retention, and defensibility. |
| `claim-analyst.md` | Breaks product directions into testable claims. |
| `uncertainty-reviewer.md` | Reviews unknowns, confidence, reversibility, and proof bar. |
| `council-facilitator.md` | Combines reviewer outputs into one verdict. |
| `registry.json` | Maps each role to its input and output schema. |
